- **Tauri v2 Docs**: https://v2.tauri.app

- **Tour of Rust**: https://tourofrust.com — step-by-step tour of Rust

## Useful Crate Discovery

- **crates.io**: https://crates.io — Rust package registry
- **lib.rs**: https://lib.rs — alternative crate discovery with better categorization
- **docs.rs**: https://docs.rs — auto-generated documentation for all published crates

## Example Tauri Applications

These open-source Tauri apps are great references for real-world patterns:

- **Tauri Examples**: https://github.com/tauri-apps/tauri/tree/dev/examples — official examples
- **ChatGPT Desktop** (unofficial): Tauri-based ChatGPT wrapper
- **Pake**: https://github.com/nicedao/pake — turn any webpage into a desktop app with Tauri
- **AppFlowy**: https://github.com/nicedao/AppFlowy — open-source Notion alternative
